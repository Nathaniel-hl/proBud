# 科研项目预算编制系统

一款基于 Vue 3 + Vite 构建的科研项目经费预算编制工具，帮助科研人员高效、准确地完成项目预算规划。

## ✨ 核心功能

### 1. 智能规划
- 内置差旅城市数据库，自动匹配城市等级与补贴标准
- 支持设备费、材料费、差旅费、国际合作费等多类别预算编制生成可行规划方案
- 智能计算各项费用，减少人工计算错误

### 2. 数据自动缓存与导出
- 表单数据实时自动保存至本地存储，刷新页面不丢失
- 支持一键导出预算明细，便于存档与提交

### 3. 金额联动，避免不一致
- 各子项费用自动汇总至预算总表
- 修改任意子项金额，总预算实时同步更新
- 确保各模块数据一致性，杜绝手工汇总误差
